# Calculator App

A simple web calculator that performs basic math operations: addition, subtraction, multiplication, division, and percentage calculations. task description: https://docs.google.com/document/d/1zpXXeSae-BlcxPKgw3DhxZA92cspVailrPYoaXSYrW8/edit?tab=t.0#heading=h.5p521hfkjq82

## How to Use

1. Enter a number - Click the number buttons (0-9) and decimal point (.)
2. Choose an operation - Click +, -, ×, ÷, or %
3. Enter another number - Click the number buttons
4. Get the result - Click = to calculate
5. Clear - Click AC to reset the calculator
6. Change sign - Click +/- to make a number positive or negative

### Examples

- `7 + 3 = 10`
- `10 - 4 = 6`
- `5 × 6 = 30`
- `15 ÷ 3 = 5`
- `6 % 8 = 0.48` (6% of 8)

## Running the App

### Development

```bash
npm install
npm run dev
```

The app will open at `http://localhost:3000`

### Production Build

```bash
npm run build
```

Built files will be in the `dist/` folder.
In js folder lies theme.js file where i made theme-choise for user to select his custom one.

# Calculator App

Простой веб-калькулятор, выполняющий основные математические операции: сложение, вычитание, умножение, деление и расчёт процентов. Описание задачи: https://docs.google.com/document/d/1zpXXeSae-BlcxPKgw3DhxZA92cspVailrPYoaXSYrW8/edit?tab=t.0#heading=h.5p521hfkjq82

## Как использовать

1. Введите число — нажмите цифровые кнопки (0–9) и десятичную точку (.).
2. Выберите операцию — нажмите +, -, ×, ÷ или %.
3. Введите другое число — нажмите цифровые кнопки.
4. Получите результат — нажмите = для вычисления.
5. Очистить — нажмите AC, чтобы сбросить калькулятор.
6. Изменить знак — нажмите +/-, чтобы сделать число положительным или отрицательным.

### Примеры

- `7 + 3 = 10`
- `10 - 4 = 6`
- `5 × 6 = 30`
- `15 ÷ 3 = 5`
- `6% 8 = 0,48` (6% от 8)

##Запуск приложения

### Начало сборки

```bash
npm install
npm run dev
```

Приложение откроется по адресу `http://localhost:3000`

### Сборка в продакшн

```bash
npm run build
```

Файлы сборки будут находиться в папке `dist/`.
В папке js лежит файл theme.js в котором я сделал выбор темы, где пользователь может настроить свою собственную кастомную тему.
